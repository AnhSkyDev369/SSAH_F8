Css Units là gì ?

Css Units hiểu đơn giản là đơn vị sử dụng trong Css giúp tuỳ chỉnh kích thước của những đối tượng hiển thị trên website

1. Absolute units

Đơn vị tuyệt đối

  • px
  • pt
  • cm
  • mm
  • inch
  • pc

2. Relative units ?

Đơn vị tuơng đối

  • %
  • rem
  • em
  • vw
  • vh
  • vmin
  • vmax
  • ex
  • ch
