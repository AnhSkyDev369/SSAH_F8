Padding CSS Là Gì ?

Sơ đồ biểu diễn dạng khối cho tất cả elements nằm trong file html Sử dụng Computed để xem sơ đồ khối trong devtools

Padding là lớp đệm.

Computed có 4 lớp lần lượt từ trong ra ngoài là lớp content,padding,border,margin.

Padding-top và padding bottom sẽ cộng giá trị px cho height .

Padding-left và padding-right sẽ cộng giá trị px cho width .
